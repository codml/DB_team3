#### 실행 명령어 : frontend 폴더에서 npm run serve 또는 yarn serve


##### git push 시 주의점!!!
###### yarn.lock 때문에 에러가 발생하니 node_modules 빼고 push 해주세요
###### git pull 에러나면 yarn.lock 삭제후 git pull하기
###### git pull 후에 yarn install 하면 정상적으로 실행됩니다
